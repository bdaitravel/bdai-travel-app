# BDAI - Better Destinations AI (Super App)

A React-based Travel Super App combining AI-curated tours, an e-commerce shop, and eSIM connectivity.

## Project Structure

```text
/
├── index.html              # Entry HTML
├── index.tsx               # Entry React Logic
├── App.tsx                 # Main Layout, Navigation & Logic
├── types.ts                # TypeScript Interfaces
├── metadata.json           # App Metadata
├── .env                    # (Create this file) API Keys
├── components/             # UI Components
│   ├── CityCard.tsx        # City Selection Card
│   ├── TourCard.tsx        # Tour List & Active Tour UI
│   ├── SchematicMap.tsx    # Leaflet Map Component
│   ├── Leaderboard.tsx     # Gamification UI
│   ├── ProfileModal.tsx    # User Profile Details
│   └── Onboarding.tsx      # Welcome Flow
├── services/
│   └── geminiService.ts    # AI Logic (Tours, Audio, Translation)
└── data/
    └── toursData.ts        # Static Data (Fallbacks & Curated Tours)
```

## Setup & Configuration

### 1. Installation
Run the following commands to install dependencies:
```bash
npm install
```

### 2. API Key Configuration (Crucial)
This app requires a Google Gemini API Key to function correctly (generate tours, translate content, create images).

1.  Get an API Key from [Google AI Studio](https://aistudio.google.com/app/apikey).
2.  Create a file named `.env` in the root directory.
3.  Add your key:
    ```env
    VITE_API_KEY=AIzaSy...YourKeyHere
    ```

### 3. Run Development Server
```bash
npm run dev
```

## Features

1.  **Explore (AI & Static):** Smart walking tours generated by Gemini or loaded from a curated database.
2.  **Passport (Profile):** Gamified user profile with levels, badges, and travel history.
3.  **Shop:** E-commerce interface for digital and physical travel goods.
4.  **Connect:** Marketplace for eSIM data plans.
5.  **Multi-language:** Full support for 10+ languages with fallback handling.
